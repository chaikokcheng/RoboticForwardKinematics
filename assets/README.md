# Assets Folder

This folder is for optional media files for the RobotRider Forward Kinematics website.

## Videos

**Note:** The website uses YouTube embeds instead of local video files.

To add your YouTube videos:

1. Upload your videos to YouTube (can be unlisted)
2. Get the video ID from the YouTube URL (e.g., from `https://www.youtube.com/watch?v=VIDEO_ID`, use `VIDEO_ID`)
3. Replace `YOUR_VIDEO_ID_HERE` in the HTML files:
   - `video1.html` - Replace with your Junior School video ID
   - `video2.html` - Replace with your High School video ID
   - `video3.html` - Replace with your Undergraduate video ID

### Example:

If your YouTube URL is: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
Then replace `YOUR_VIDEO_ID_HERE` with: `dQw4w9WgXcQ`

## Images (Optional)

You can add images in the `images/` folder if needed for other purposes:

- Images: JPG or PNG format

## Note

The website will work once you add your YouTube video IDs to the HTML files. No local video files are required.
